# 0001: Use The Name Bloom

Date: 2026-06-01

## Decision

Use `bloom` as the repository and product name.

## Context

The project started from Extender UI/backend work, but the target is more generic than Extender: a lab-friendly configurable web interface for robot teleoperation, supervision, and device control.

`bloom` keeps the nature-inspired direction and leaves room for reuse across ISIR projects.

## Consequences

- User-facing docs and package names should use Bloom.
- Extender remains one deployment/consumer, not the identity of the whole project.
- Legacy references to `extender-web-interface` should be treated as outdated planning notes.
